# adventofcode-2025

Solutions for [Advent of Code 2025](https://adventofcode.com/2025), presented in [Java 25](https://www.baeldung.com/java-25-features) and built using [Gradle 9.2.1](https://docs.gradle.org/9.2.1/userguide/userguide.html).

# Solutions
The solution for each day is in its own class under `/src/main/java/com/jacoblucas/adventofcode2025`. Each respective day has its own main method to run the solution for that day.

The solutions for parts 1 and 2 for the day will be printed to stdout.
